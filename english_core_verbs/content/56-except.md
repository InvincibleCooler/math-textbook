# except

> '~을 제외하고' — 배제

## 감각
전체에서 하나를 빼는 감각.
- Everyone came except Tom.
- I like all fruit except bananas.
- The store is open every day except Sunday.

## except vs besides
- **except**: 제외 — Everyone except me. (나만 빼고)
- **besides**: 추가 — Besides me, she invited Tom. (나 외에 Tom도)

## except vs except for
- **except**: 주어 뒤 직접 — Everyone except Tom.
- **except for**: 문장 앞·중간 — Except for the ending, the movie was great.

## 함정
- (X) except of → (O) except / except for

## 연습
1. I eat everything _____ seafood. (except / besides)
2. _____ the noise, the hotel was perfect. (Except for / Except)

**정답**: 1. except  2. Except for
