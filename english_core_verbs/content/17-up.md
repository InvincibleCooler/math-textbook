# up

> '위로' 올라간다 — 그리고 끝까지 올라가면 '완료'가 된다

## 감각 확장: 위로 → 완료 → 증가 → 출현 → 시작

### 위로: 물리적 상승
아래에서 위로 올라가는 감각.
- Stand up, please.
- Look up at the sky.

### 완료: 끝까지
위로 올라가서 꼭대기에 도달 → 다 끝남. **네이티브가 가장 많이 쓰는 감각.**
- Eat up your vegetables. (다 먹어)
- Drink up. We have to go. (다 마셔)
- Use up all the paper. (다 써버리다)
- Time's up! (시간 끝!)

### 증가: 올라감
수치·강도가 위로 올라가는 감각.
- Turn up the volume.
- Speed up. We're late.

### 출현: 나타남
아래에서 위로 올라와 보이는 감각.
- Something came up. (일이 생겼어)
- She showed up late. (나타났다)

### 시작·준비
아래에서 일으켜 세우는 감각.
- Set up the equipment.
- Warm up before you exercise.

### 분할·해체
조각조각 올려서 분리하는 감각.
- Break up the chocolate into pieces.
- They broke up last month. (헤어지다)

## 동사 조합 TOP
| 동사 + up | 감각 | 예문 |
|-----------|------|------|
| get up | 일어나다 | I get up at 6 every day. |
| give up | 포기하다 | Don't give up. |
| make up | 화해 / 구성 / 꾸미다 | Let's make up. |
| come up | 다가오다 / 생기다 | A problem came up. |
| bring up | 화제 꺼내다 / 키우다 | Don't bring up that topic. |
| keep up | 따라가다 | Keep up the good work. |
| turn up | 나타나다 / 올리다 | He finally turned up. |
| hold up | 지연시키다 / 버티다 | What's holding things up? |

## vs out (완료 감각 비교)
- **up**: 끝까지 채워서 완료 — eat up (다 먹다), fill up (다 채우다)
- **out**: 다 빠져나가서 완료 — run out (다 떨어지다), burn out (다 타다)

## 함정
- (X) Wake up! → (O) 맞음. 가장 자연스러운 표현.
- (X) I showed up to late. → (O) I showed up late. (to 불필요)
- (X) Pick up it. → (O) Pick it up. (대명사는 사이에)

## 연습
1. Hurry _____! We're going to miss the bus. (up / on)
2. She _____ up a great idea. (came up with / made up with)
3. Time's _____. Put down your pens. (up / out)
4. He _____ up late to the meeting again. (showed / turned) — 둘 다 가능
5. _____ up before you run. (Warm / Heat)

**정답**: 1. up  2. came up with  3. up  4. showed/turned  5. Warm
