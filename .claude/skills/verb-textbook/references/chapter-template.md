# 챕터 템플릿

**핵심 원칙: 짧고 핵심만. 외우지 말고 느끼게.**

## 동사 챕터 템플릿 (Part 1)

```markdown
# [동사]

> (핵심 감각 한 문장)

## 감각 지도

### 1. [가장 빈도 높은 의미] ★★★★★
(한 줄 설명)
- 예문 2개 (일상 대화)

### 2. [두 번째 의미] ★★★★
(한 줄 설명)
- 예문 2개

### 3. [세 번째 의미] ★★★
...
(빈도순으로 주요 의미만. 마이너한 의미는 생략)

## 필수 패턴
| 패턴 | 감각 | 예문 |
|------|------|------|
(가장 많이 쓰는 5~8개만)

## 함정
- (X) → (O) (한국인 오류 2~3개)

## 연습
(빈칸 5문제 + 정답)
```

## 전치사 챕터 템플릿 (Part 2)

```markdown
# [전치사]

> (핵심 공간 감각 한 문장)

## 감각 확장: 공간 → 시간 → 추상

### 공간
(한 줄 설명 + 예문 2개)

### 시간 (해당 시)
(한 줄 설명 + 예문 2개)

### 추상
(한 줄 설명 + 예문 2개)

## 동사 조합 TOP
| 동사 + [전치사] | 감각 | 예문 |
|----------------|------|------|
(Part 1 동사와의 핵심 조합 5~8개)

## vs [반대/유사 전치사]
(핵심 차이 한 줄 + 비교 예문 2개)

## 함정
- (X) → (O) (2~3개)

## 연습
(빈칸 5문제 + 정답)
```

## 분량 가이드
- 동사 챕터: A4 기준 2~3장
- 전치사 챕터: A4 기준 1~2장
- 불필요한 문법 설명 배제
- 예문은 짧고 실용적인 것만
- "이것만 알면 된다" 느낌
