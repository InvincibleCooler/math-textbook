# Treaty Summary - 조약 요약 카드 생성

개별 조약을 수능 학습에 최적화된 요약 카드 형태로 정리하는 스킬.

## 트리거
- "~조약 정리해줘", "~조약 알려줘"
- 특정 조약명이 언급될 때

## 작업 절차

1. 조약명을 확인하고, 정식 명칭과 통칭을 모두 파악
2. 아래 템플릿에 맞춰 조약 정보를 작성
3. 수능 기출 또는 출제 가능성이 높은 포인트를 별도 강조
4. 관련 조약이 있으면 하단에 연결 정보 추가

## 출력 템플릿

```markdown
---

### [조약명] ([연도])

> **한줄 요약**: [이 조약을 한 문장으로 설명]

**체결 장소**: [도시, 국가]

#### 참여국
| 당시 국호 | 현재 국명 | 역할/입장 |
|-----------|-----------|-----------|
| 예: 프로이센 | 독일 | 주도국 |

#### 왜 이 조약이 생겼나?
- [배경 1: 전쟁이나 사건]
- [배경 2: 관련 갈등]
- [직접적 계기]

#### 핵심 내용
1. [조항/결정 1]
2. [조항/결정 2]
3. [조항/결정 3]

#### 이후 어떻게 됐나?
- [단기적 영향]
- [장기적 영향]
- [이 조약이 깨지거나 발전한 계기]

#### 수능 키워드
`키워드1` `키워드2` `키워드3`

> 관련 조약: [앞뒤로 연결되는 조약명]

---
```

## 작성 규칙

1. **국호**: 반드시 당시 국호를 먼저 쓰고 괄호 안에 현재 국명 병기
   - 예: 프로이센(현 독일), 오스만 제국(현 튀르키예), 청(현 중국)
2. **연도**: 서기 연도 사용, 필요시 음력/양력 구분
3. **배경**: "왜?"에 답하는 형태로, 인과관계를 명확히
4. **핵심 내용**: 최대 5개, 시험에 나올 만한 것 우선
5. **영향**: 다음 조약이나 사건으로의 연결고리 포함
6. **수능 키워드**: 선택지나 지문에 자주 등장하는 단어/구문
7. **분량**: 조약 1개당 A4 반 페이지~1페이지 분량
8. **어투**: 평서문, 존댓말 없이 간결하게 (~다, ~했다)
