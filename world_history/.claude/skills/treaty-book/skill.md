# Treaty Book - 세계사 조약 모음집 (오케스트레이터)

수능 세계사에 출제되는 주요 조약들을 시대별로 엮어 한 권의 간결한 책으로 만드는 스킬.

## 트리거
- "조약 책 만들어줘", "조약 모음 정리"
- "시대별 조약 정리", "전체 조약 요약"
- 여러 조약을 한꺼번에 정리 요청 시

## 워크플로우

### Step 1: 범위 확인
사용자에게 확인:
- 전체 시대? 특정 시대? (근세/근대/제국주의/1차대전/2차대전/냉전)
- 특정 지역? (유럽/동아시아/중동 등)
- 출력 형식? (마크다운 파일 / 화면 출력)

### Step 2: 조약 목록 구성
시대별로 수능 빈출 조약을 선정:

#### 근세 (15~17세기)
- 토르데시야스 조약 (1494)
- 베스트팔렌 조약 (1648)

#### 근대 초기 (18~19세기 초)
- 위트레흐트 조약 (1713)
- 파리 조약 (1763)
- 빈 의정서/빈 회의 (1815)

#### 제국주의 시대 (19세기 중후반)
- 난징 조약 (1842)
- 톈진 조약 (1858)
- 베이징 조약 (1860)
- 시모노세키 조약 (1895)
- 베를린 회의 (1884~1885)
- 포츠머스 조약 (1905)

#### 제1차 세계대전 전후
- 베르사유 조약 (1919)
- 생제르맹 조약 (1919)
- 트리아농 조약 (1920)
- 세브르 조약 (1920) / 로잔 조약 (1923)
- 워싱턴 해군 군축 조약 (1922)
- 로카르노 조약 (1925)

#### 제2차 세계대전 전후
- 뮌헨 협정 (1938)
- 독소 불가침 조약 (1939)
- 대서양 헌장 (1941)
- 카이로 선언 (1943)
- 얄타 회담 (1945)
- 포츠담 선언 (1945)
- 샌프란시스코 강화 조약 (1951)

#### 냉전과 현대
- 제네바 협정 (1954)
- 바르샤바 조약 (1955)
- 핵확산금지조약/NPT (1968)
- 파리 평화 협정 (1973)
- 헬싱키 협정 (1975)
- INF 조약 (1987)

### Step 3: 개별 조약 작성
- `treaty-summary` 스킬의 템플릿을 사용하여 각 조약 작성
- `treaty-analyst` 에이전트의 분석 프레임워크 적용

### Step 4: 책 구성
아래 구조로 최종 문서 조립:

```markdown
# 수능 세계사 조약 한눈에 보기

## 이 책의 사용법
- 시대순으로 읽으면 역사의 흐름이 보입니다
- 각 조약의 "수능 키워드"를 먼저 체크하세요
- 관련 조약 링크를 따라가면 연결고리가 보입니다

## 조약 연대표 (타임라인)
[전체 조약을 연도순 한줄 목록으로]

---

## 1장. 근세의 조약 (15~17세기)
[조약 카드들]

## 2장. 근대 초기의 조약 (18~19세기 초)
[조약 카드들]

## 3장. 제국주의 시대의 조약 (19세기 중후반)
[조약 카드들]

...

## 부록: 수능 키워드 색인
[키워드 → 조약 매핑 테이블]
```

### Step 5: 출력
- 사용자가 파일 저장을 원하면 `.md` 파일로 저장
- 화면 출력 시 장(chapter) 단위로 나눠서 출력
- 한 번에 전체를 출력하면 너무 길므로, 시대별로 나눠서 진행

## 출력 규칙
- 전체 분량: 조약 1개당 반~1페이지, 총 20~30페이지 수준
- 시대별 도입부에 해당 시대 배경 2~3문장 추가
- 조약 간 인과관계 화살표나 참조 표시 필수
- 마지막에 키워드 색인 테이블 포함
