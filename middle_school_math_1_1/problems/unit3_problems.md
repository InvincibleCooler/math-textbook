### 수행평가 1: 생활 속 규칙을 문자식으로 표현하기

**과제 유형**: 포트폴리오 | **성취기준**: [9수02-01], [9수02-02], [9수02-03]

#### 과제 설명

일상생활에서 수량 사이의 관계를 찾아 문자를 사용한 식으로 표현하고, 식의 값을 구하는 활동을 수행하시오.

#### 수행 내용

1. **주제 선정**: 일상생활에서 두 양 사이의 관계가 있는 상황을 **3가지 이상** 찾으시오.
   - 예: 통신요금, 교통카드 잔액, 문구점 구매, 저금통 저축 등

2. **문자식 표현**: 각 상황을 문자를 사용한 식으로 나타내시오.
   - 사용할 문자가 무엇을 의미하는지 명확히 정의할 것

3. **식의 값 구하기**: 문자에 구체적인 수를 대입하여 식의 값을 구하시오.
   - 각 상황에서 최소 **2개 이상**의 서로 다른 값을 대입할 것

4. **일차식 계산 활용**: 찾은 상황 중 **1가지 이상**에서 일차식의 덧셈 또는 뺄셈이 활용되는 경우를 만들어 계산 과정을 보이시오.
   - 예: A 요금제의 한 달 비용과 B 요금제의 한 달 비용의 차이

5. **보고서 작성**: 위 활동을 정리하여 보고서를 작성하시오.

#### 평가 기준

| 평가 항목 | 배점 | 평가 내용 |
|-----------|------|-----------|
| 주제 선정의 적절성 | 20점 | 실생활과 연관된 적절한 상황을 선정하였는가? |
| 문자식 표현의 정확성 | 30점 | 수량 관계를 문자식으로 정확하게 표현하였는가? |
| 식의 값 계산 | 20점 | 대입 과정과 계산이 정확한가? |
| 일차식 계산 활용 | 20점 | 일차식의 덧셈/뺄셈을 적절히 활용하였는가? |
| 보고서 완성도 | 10점 | 보고서가 체계적이고 논리적으로 작성되었는가? |

### 수행평가 2: 문자식 퀴즈 만들기 (모둠 활동)

**과제 유형**: 프로젝트 | **성취기준**: [9수02-01], [9수02-02], [9수02-03]

#### 과제 설명

모둠별로 "문자의 사용과 식의 계산" 단원의 내용을 활용한 퀴즈를 만들고, 다른 모둠과 서로 풀어보는 활동을 수행하시오.

#### 수행 내용

1. **퀴즈 제작** (모둠별 $5$문제 이상):
   - 문자를 사용한 식 표현 문제: 최소 $1$문제
   - 식의 값 구하기 문제: 최소 $1$문제
   - 일차식의 계산 문제: 최소 $1$문제
   - 실생활 상황을 활용한 문제: 최소 $1$문제
   - 창의적이거나 재미있는 문제: 최소 $1$문제

2. **정답 및 풀이 작성**: 모든 문제에 대해 모범 풀이를 작성하시오.

3. **교차 풀이**: 다른 모둠이 제작한 퀴즈를 풀고, 그 결과를 공유하시오.

4. **상호 피드백**: 다른 모둠의 문제에 대해 좋은 점과 개선할 점을 작성하시오.

#### 평가 기준

| 평가 항목 | 배점 | 평가 내용 |
|-----------|------|-----------|
| 문제의 수학적 정확성 | 30점 | 문제의 조건이 충분하고 정답이 정확한가? |
| 문제 유형의 다양성 | 20점 | 다양한 성취기준과 유형을 포함하였는가? |
| 풀이의 완성도 | 20점 | 단계별 풀이가 논리적이고 명확한가? |
| 창의성 | 20점 | 독창적이고 흥미로운 문제가 포함되어 있는가? |
| 협력 및 피드백 | 10점 | 모둠원이 고르게 참여하고 건설적인 피드백을 제공하였는가? |
