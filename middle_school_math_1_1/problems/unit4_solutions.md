## 중단원 1 연습문제 풀이: 방정식과 등식의 성질

---

### 1번

**정답**: (나), (다), (마)

**풀이**:

- (가) $3x + 5$: 등호($=$)가 없으므로 등식이 아니다. (식)
- (나) $2x - 1 = 7$: 등호가 있으므로 등식이다.
- (다) $4 + 6 = 10$: 등호가 있으므로 등식이다.
- (라) $x^2 > 0$: 부등호($>$)가 사용되었으므로 등식이 아니다. (부등식)
- (마) $\dfrac{x}{3} = x - 2$: 등호가 있으므로 등식이다.

**흔한 오답**: (라)를 등식으로 착각하는 경우. 등식은 반드시 등호($=$)로 연결된 식이다.

---

### 2번

**정답**: 방정식 -- (나), (라) / 항등식 -- (가), (다)

**풀이**:

- (가) $2(x + 3) = 2x + 6$: 좌변을 전개하면 $2x + 6 = 2x + 6$으로, $x$에 어떤 값을 대입해도 항상 성립한다. -- **항등식**
- (나) $5x - 4 = 11$: $x = 3$일 때만 성립한다. -- **방정식**
- (다) $x + 0 = x$: $x$에 어떤 값을 대입해도 항상 성립한다. -- **항등식**
- (라) $3x = x + 8$: $x = 4$일 때만 성립한다. -- **방정식**

**흔한 오답**: (가)를 방정식으로 판단하는 경우. 좌변을 전개하여 양변이 같은 식인지 확인해야 한다.

---

### 3번

**정답**: (1) 해이다 (2) 해가 아니다 (3) 해이다

**풀이**:

(1) $3 \times (-2) + 10 = -6 + 10 = 4$ (우변과 같다) $\rightarrow$ 해이다.

(2) $2 \times (-2) - 5 = -4 - 5 = -9 \neq -1$ $\rightarrow$ 해가 아니다.

(3) $-(-2) + 7 = 2 + 7 = 9$ (우변과 같다) $\rightarrow$ 해이다.

**흔한 오답**: (3)에서 $-x$에 $x = -2$를 대입할 때 $-(-2) = -2$로 잘못 계산하는 경우.

---

### 4번

**정답**: $x = 3$

**풀이**:

- $x = 1$: $4(1) - 3 = 1 \neq 9$
- $x = 2$: $4(2) - 3 = 5 \neq 9$
- $x = 3$: $4(3) - 3 = 9$ (성립)
- $x = 4$: $4(4) - 3 = 13 \neq 9$

따라서 해는 $x = 3$이다.

**흔한 오답**: 대입 계산에서 부호 실수를 하는 경우.

---

### 5번

**정답**: (1) $5$ (2) $3$ (3) $4$ (4) $6$

**풀이**:

(1) 등식의 양변에 같은 수를 더해도 등식이 성립하므로 $\square = 5$

(2) 등식의 양변에서 같은 수를 빼도 등식이 성립하므로 $\square = 3$

(3) 등식의 양변에 같은 수를 곱해도 등식이 성립하므로 $\square = 4$

(4) 등식의 양변을 같은 수($0$이 아닌)로 나누어도 등식이 성립하므로 $\square = 6$

**흔한 오답**: 등식의 성질을 양변 각각에 다른 수를 적용하는 것으로 오해하는 경우.

---

### 6번

**정답**: (1) $x = 6$ (2) $x = -3$ (3) $x = -6$ (4) $x = -10$

**풀이**:

(1) $x + 9 = 15$의 양변에서 $9$를 빼면 $x = 15 - 9 = 6$

(2) $x - 4 = -7$의 양변에 $4$를 더하면 $x = -7 + 4 = -3$

(3) $3x = -18$의 양변을 $3$으로 나누면 $x = \dfrac{-18}{3} = -6$

(4) $\dfrac{x}{5} = -2$의 양변에 $5$를 곱하면 $x = -2 \times 5 = -10$

**흔한 오답**: (2)에서 양변에서 $4$를 빼는 실수, (4)에서 $5$로 나누는 실수.

---

### 7번

**정답**: (나), (다)

**풀이**:

- (가) $a = b$이면 양변에 같은 수 $c$를 더하여 $a + c = b + c$이다. $a + c = b - c$가 성립하려면 $c = -c$, 즉 $c = 0$이어야 하므로 일반적으로 거짓이다. -- **거짓**
- (나) $a + 3 = b + 3$이면 양변에서 $3$을 빼면 $a = b$이다. -- **참**
- (다) $2a = 2b$이면 양변을 $2$로 나누면 $a = b$이다. -- **참**
- (라) $0$으로 나누는 것은 정의되지 않으므로 $\dfrac{a}{0}$은 의미가 없다. -- **거짓**

**흔한 오답**: (라)에서 $\dfrac{a}{0} = \dfrac{b}{0}$이 성립한다고 판단하는 경우. 0으로 나누기는 불가능하다.

---

### 8번

**정답**: $x = 75$ (물체 1개의 무게는 $75$ g)

**풀이**:

양팔저울이 수평이므로: $2x + 50 = 200$

등식의 양변에서 $50$을 빼면: $2x = 150$

등식의 양변을 $2$로 나누면: $x = 75$

따라서 물체 1개의 무게는 $75$ g이다.

**흔한 오답**: 등식을 $2x = 200 + 50 = 250$으로 세우는 경우. 왼쪽 접시와 오른쪽 접시의 무게를 정확히 파악해야 한다.

---

### 9번

**정답**: $x = 6$

**풀이**:

"$x$에 $7$을 더한 값" $= x + 7$

"$x$의 $3$배에서 $5$를 뺀 값" $= 3x - 5$

두 값이 같으므로: $x + 7 = 3x - 5$

양변에서 $x$를 빼면: $7 = 2x - 5$

양변에 $5$를 더하면: $12 = 2x$

양변을 $2$로 나누면: $x = 6$

**검산**: $6 + 7 = 13$, $3 \times 6 - 5 = 13$ (성립)

**흔한 오답**: "3배에서 5를 뺀 값"을 $3(x - 5)$로 세우는 경우.

---

### 10번

**정답**: $a = 2$

**풀이**:

$x = 4$를 $ax + 2 = 10$에 대입하면

$4a + 2 = 10$

$4a = 8$

$a = 2$

**흔한 오답**: $a$를 미지수, $x$를 상수로 혼동하여 방정식을 잘못 세우는 경우.

---

### 11번

**정답**: (가) 항등식, (나) 방정식

**풀이**:

(가) 좌변 $= 3(x - 1) + 3 = 3x - 3 + 3 = 3x$, 우변 $= 3x$

좌변 $=$ 우변이므로 $x$에 어떤 값을 대입해도 항상 성립한다. 따라서 **항등식**이다.

(나) 좌변 $= 2(x + 4) = 2x + 8$, 우변 $= 3x + 1$

$2x + 8 = 3x + 1$을 정리하면 $x = 7$일 때만 성립한다. 따라서 **방정식**이다.

**이유**: 항등식은 미지수에 어떤 값을 대입해도 성립하는 등식이고, 방정식은 특정 값에서만 성립하는 등식이다. 좌변을 전개/정리하여 우변과 비교하면 판별할 수 있다.

**흔한 오답**: 전개하지 않고 겉모습만 보고 판단하는 경우.

## 중단원 2 연습문제 풀이: 일차방정식의 풀이와 활용

---

### 12번

**정답**: (1) $x = 3$ (2) $x = 3$ (3) $x = -3$ (4) $x = 4$

**풀이**:

(1) $5x - 3 = 12$
$$5x = 12 + 3 = 15$$
$$x = 3$$

(2) $-2x + 7 = 1$
$$-2x = 1 - 7 = -6$$
$$x = 3$$

(3) $4x + 9 = x$
$$4x - x = -9$$
$$3x = -9$$
$$x = -3$$

(4) $6x - 5 = 2x + 11$
$$6x - 2x = 11 + 5$$
$$4x = 16$$
$$x = 4$$

**흔한 오답**: 이항할 때 부호를 바꾸지 않는 실수. 예: (3)에서 $9$를 이항할 때 $+9$로 옮기는 경우.

---

### 13번

**정답**: (1) $x = 5$ (2) $x = -17$ (3) $x = -6$

**풀이**:

(1) $3(x - 2) = 9$
$$3x - 6 = 9$$
$$3x = 15$$
$$x = 5$$

(2) $2(3x + 1) = 5(x - 3)$
$$6x + 2 = 5x - 15$$
$$6x - 5x = -15 - 2$$
$$x = -17$$

(3) $4(x + 5) - 3(x - 2) = 20$
$$4x + 20 - 3x + 6 = 20$$
$$x + 26 = 20$$
$$x = -6$$

**흔한 오답**: (3)에서 $-3(x - 2)$를 전개할 때 $-3x - 6$으로 계산하는 경우. 음수 $\times$ 음수 $=$ 양수이므로 $-3 \times (-2) = +6$이다.

---

### 14번

**정답**: (1) $x = 5$ (2) $x = 3$ (3) $x = 20$

**풀이**:

(1) $0.3x + 1.2 = 2.7$

양변에 $10$을 곱하면: $3x + 12 = 27$
$$3x = 15$$
$$x = 5$$

(2) $0.5x - 0.2 = 0.1x + 1$

양변에 $10$을 곱하면: $5x - 2 = x + 10$
$$4x = 12$$
$$x = 3$$

(3) $0.04x + 0.7 = 0.1x - 0.5$

양변에 $100$을 곱하면: $4x + 70 = 10x - 50$
$$-6x = -120$$
$$x = 20$$

**흔한 오답**: 양변에 곱하는 수를 잘못 정하는 경우. 소수점 이하 자릿수가 가장 많은 항에 맞추어 곱해야 한다.

---

### 15번

**정답**: (1) $x = 4$ (2) $x = \dfrac{19}{5}$ (3) $x = \dfrac{23}{3}$

**풀이**:

(1) $\dfrac{x}{2} + 3 = 5$

양변에 $2$를 곱하면: $x + 6 = 10$
$$x = 4$$

(2) $\dfrac{2x - 1}{3} = \dfrac{x + 5}{4}$

양변에 $12$ (분모 $3, 4$의 최소공배수)를 곱하면: $4(2x - 1) = 3(x + 5)$
$$8x - 4 = 3x + 15$$
$$5x = 19$$
$$x = \dfrac{19}{5}$$

(3) $\dfrac{x + 3}{2} - \dfrac{x - 1}{5} = 4$

양변에 $10$ (분모 $2, 5$의 최소공배수)을 곱하면: $5(x + 3) - 2(x - 1) = 40$
$$5x + 15 - 2x + 2 = 40$$
$$3x + 17 = 40$$
$$3x = 23$$
$$x = \dfrac{23}{3}$$

**흔한 오답**: 분모의 최소공배수를 구할 때 실수하거나, 양변에 곱한 후 분배법칙 적용을 빠뜨리는 경우.

---

### 16번

**정답**: $17, \, 19, \, 21$

**풀이**:

연속하는 세 홀수를 $x, \, x + 2, \, x + 4$로 놓으면

$$x + (x + 2) + (x + 4) = 57$$
$$3x + 6 = 57$$
$$3x = 51$$
$$x = 17$$

따라서 세 홀수는 $17, \, 19, \, 21$이다.

**검산**: $17 + 19 + 21 = 57$ (성립)

**흔한 오답**: 연속하는 세 홀수를 $x, \, x + 1, \, x + 2$로 놓는 경우. 홀수는 $2$씩 차이가 난다.

---

### 17번

**정답**: 귤 $6$개, 사과 $4$개

**풀이**:

귤을 $x$개 샀다고 하면, 사과는 $(10 - x)$개이다.

$$800x + 1500(10 - x) = 10800$$
$$800x + 15000 - 1500x = 10800$$
$$-700x = -4200$$
$$x = 6$$

따라서 귤 $6$개, 사과 $10 - 6 = 4$개이다.

**검산**: $800 \times 6 + 1500 \times 4 = 4800 + 6000 = 10800$원 (성립)

**흔한 오답**: 합계 개수 조건을 빠뜨리고 미지수를 $2$개 놓으려는 경우.

---

### 18번

**정답**: 걸은 거리 $1.2$ km, 뛴 거리 $1.2$ km

**풀이**:

걸은 시간을 $x$시간이라 하면, 뛴 시간은 $\left(\dfrac{24}{60} - x\right) = \left(\dfrac{2}{5} - x\right)$시간이다.

(거리) = (속력) $\times$ (시간)이므로

$$4x + 12\left(\dfrac{2}{5} - x\right) = 2.4$$

$$4x + \dfrac{24}{5} - 12x = 2.4$$

$$-8x + 4.8 = 2.4$$

$$-8x = -2.4$$

$$x = 0.3 \text{ (시간)} = 18 \text{ (분)}$$

이것은 총 시간 $24$분보다 작아야 하므로, 검산하면:

걸은 시간 $0.3$시간 $= 18$분, 뛴 시간 $= 24 - 18 = 6$분 $= 0.1$시간

걸은 거리 $= 4 \times 0.3 = 1.2$ km

뛴 거리 $= 12 \times 0.1 = 1.2$ km

합계 $= 1.2 + 1.2 = 2.4$ km (성립)

**흔한 오답**: 단위를 통일하지 않는 경우. 시속(km/h)과 분(min)을 같은 식에 사용하면 오류가 발생한다. 시간을 "시간" 단위로 통일하거나, 속력을 "분속"으로 변환해야 한다.

### 19번

**정답**: $180$ g

**풀이**:

더 넣을 물의 양을 $x$ g이라 하자.

$8$ %의 소금물 $300$ g에 들어 있는 소금의 양: $300 \times \dfrac{8}{100} = 24$ (g)

물을 $x$ g 더 넣으면 소금물의 총 양은 $(300 + x)$ g이고, 소금의 양은 변하지 않으므로

$$\dfrac{24}{300 + x} = \dfrac{5}{100}$$

$$24 \times 100 = 5(300 + x)$$

$$2400 = 1500 + 5x$$

$$5x = 900$$

$$x = 180$$

**검산**: 소금물 $300 + 180 = 480$ g, 소금 $24$ g이므로 농도 $= \dfrac{24}{480} \times 100 = 5$ (%) (성립)

**흔한 오답**: 물을 넣을 때 소금의 양도 변한다고 생각하는 경우. 물만 추가하면 소금의 양은 일정하다.

### 20번

**정답**: $14$년 후

**풀이**:

$x$년 후에 어머니의 나이가 딸의 나이의 $2$배가 된다고 하면

$$42 + x = 2(14 + x)$$
$$42 + x = 28 + 2x$$
$$42 - 28 = 2x - x$$
$$14 = x$$

따라서 $14$년 후이다.

**검산**: $14$년 후 어머니 $42 + 14 = 56$세, 딸 $14 + 14 = 28$세, $56 = 2 \times 28$ (성립)

**흔한 오답**: "$x$년 후"를 나이에 더하지 않고 현재 나이로만 방정식을 세우는 경우.

---

### 21번

**정답**: $15$

**풀이**:

자연수를 $x$라 하자.

"$x$의 $3$배에서 $7$을 뺀 값"$= 3x - 7$

"$x$의 $2$배보다 $8$만큼 큰 값"$= 2x + 8$

$$3x - 7 = 2x + 8$$
$$x = 15$$

**검산**: $3 \times 15 - 7 = 38$, $2 \times 15 + 8 = 38$ (성립). $15$는 자연수이므로 조건을 만족한다.

**흔한 오답**: "8만큼 크다"를 $2x - 8$로 세우는 경우.

### 22번

**정답**: 가로 $12$ cm, 세로 $8$ cm

**풀이**:

세로의 길이를 $x$ cm라 하면, 가로의 길이는 $(x + 4)$ cm이다.

직사각형의 둘레 $= 2(\text{가로} + \text{세로})$이므로

$$2\{(x + 4) + x\} = 40$$
$$2(2x + 4) = 40$$
$$2x + 4 = 20$$
$$2x = 16$$
$$x = 8$$

따라서 세로 $= 8$ cm, 가로 $= 8 + 4 = 12$ cm이다.

**검산**: $2(12 + 8) = 2 \times 20 = 40$ cm (성립)

**흔한 오답**: 둘레 공식을 $\text{가로} + \text{세로} = 40$으로 세우는 경우.

---

### 23번

**정답**: $25$분 후

**풀이**:

출발한 지 $x$분 후에 형이 동생보다 $500$ m 앞서게 된다고 하자.

형의 이동 거리: $80x$ m, 동생의 이동 거리: $60x$ m

$$80x - 60x = 500$$
$$20x = 500$$
$$x = 25$$

따라서 $25$분 후이다.

**검산**: 형 $80 \times 25 = 2000$ m, 동생 $60 \times 25 = 1500$ m, 차이 $= 500$ m (성립)

**흔한 오답**: 형의 속력에서 동생의 속력을 빼지 않고 따로 방정식을 세우려다 복잡해지는 경우.

### 24번

**정답**: $a = 20$

**풀이**:

정가 $25000$원에서 $a$ % 할인하면 판매가는

$$25000 - 25000 \times \dfrac{a}{100} = 20000$$

$$25000 - 250a = 20000$$

$$-250a = -5000$$

$$a = 20$$

따라서 할인율은 $20$ %이다.

**검산**: $25000 \times (1 - 0.2) = 25000 \times 0.8 = 20000$원 (성립)

**흔한 오답**: $25000 \times a = 20000$으로 방정식을 세우는 경우. 할인율은 백분율이므로 $\dfrac{a}{100}$을 곱해야 한다.

### 25번

**정답**: $x = -17$

**풀이**:

$$\dfrac{3x + 1}{4} - \dfrac{2(x - 3)}{3} = \dfrac{5}{6}$$

분모 $4, 3, 6$의 최소공배수 $12$를 양변에 곱하면

$$3(3x + 1) - 4 \cdot 2(x - 3) = 2 \times 5$$

$$9x + 3 - 8(x - 3) = 10$$

$$9x + 3 - 8x + 24 = 10$$

$$x + 27 = 10$$

$$x = -17$$

**검산**: $\dfrac{3(-17) + 1}{4} - \dfrac{2(-17 - 3)}{3} = \dfrac{-50}{4} + \dfrac{40}{3} = -\dfrac{25}{2} + \dfrac{40}{3} = \dfrac{-75 + 80}{6} = \dfrac{5}{6}$ (성립)

**흔한 오답**: 최소공배수를 곱할 때 각 항에 빠짐없이 곱하지 않는 경우. 특히 $\dfrac{2(x-3)}{3}$ 앞의 $-$ 부호를 놓치기 쉽다.

---

### 26번

**정답**: $a = -2$

**풀이**:

$x = -1$을 $3(x - a) = 2x + 5$에 대입하면

$$3(-1 - a) = 2(-1) + 5$$
$$-3 - 3a = 3$$
$$-3a = 6$$
$$a = -2$$

**검산**: $3(-1 - (-2)) = 3(1) = 3$, $2(-1) + 5 = 3$ (성립)

**흔한 오답**: $x - a$를 계산할 때 괄호를 빠뜨려 $3x - a = 2x + 5$로 전개하는 경우.
