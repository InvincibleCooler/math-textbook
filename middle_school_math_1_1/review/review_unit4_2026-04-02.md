## 치명적 이슈

### [C-01] 문제 41번 정답 범위 오류 (unit4_solutions.md, 985~1013행)

**문제**: "방정식 $5(x-2)+a=3x+7$의 해가 자연수가 되도록 하는 **정수** $a$의 값을 **모두** 구하시오."

**현재 정답**: $a = -1, 1, 3, 5, 7, 9, \ldots$ (15 이하의 모든 홀수)로 무한히 많은 답을 제시.

**문제점**: "모두 구하시오"라고 했지만 정수 $a$의 값이 무한히 많아($a \le 15$인 모든 홀수) 답을 "모두" 나열할 수 없다. 문제의 조건이 불충분하여 유한한 답을 도출할 수 없다.

**수정안**: 다음 중 택 1.
- (방안 1) "**양의** 정수 $a$의 값을 모두 구하시오" → 답: $a = 1, 3, 5, 7, 9, 11, 13, 15$ (8개)
- (방안 2) "$1 \le a \le 10$인 정수 $a$의 값을 모두 구하시오" → 답: $a = 1, 3, 5, 7, 9$ (5개)
- (방안 3) "정수 $a$의 값의 **개수**를 구하시오"라고 하되 범위를 명시

### [C-02] 본문 중단원 4-2 마무리 문제 8번 -- 답이 분수로 나오는 문제 설계 (4-2_중단원마무리.md, 65행)

**문제**: "분속 $60\,\text{m}$로 걷다가 분속 $200\,\text{m}$로 뛰어서 총 $20$분 만에 $2.4\,\text{km}$에 도착하였습니다. 걸은 거리와 뛴 거리를 각각 구하시오."

**검산**: 걸은 시간 $x$분이라 하면:
$$60x + 200(20-x) = 2400$$
$$60x + 4000 - 200x = 2400$$
$$-140x = -1600$$
$$x = \frac{80}{7} \approx 11.43\text{분}$$

걸은 거리 $= 60 \times \frac{80}{7} = \frac{4800}{7} \approx 685.7\,\text{m}$, 뛴 거리 $= \frac{12000}{7} \approx 1714.3\,\text{m}$

**문제점**: 답이 $\frac{4800}{7}$m, $\frac{12000}{7}$m으로 비정수 분수이다. 서술형 문제로 풀이 과정을 요구하는 취지에서는 수학적으로 오류는 아니나, 실생활 맥락에서 "685.7m를 걸었다"는 답은 부자연스럽고, 중1 학생이 검산하기 어렵다. **깔끔한 정수 답이 나오도록 수치를 수정해야 한다.**

**수정안**: 분속 $60\,\text{m}$, 분속 $200\,\text{m}$, 총 **$20$분**, 거리 $2.4\,\text{km}$를 다음과 같이 조정:
- 분속 $80\,\text{m}$, 분속 $200\,\text{m}$, 총 $20$분, 거리 $2.8\,\text{km}$ → 걸은 시간 $10$분, 뛴 시간 $10$분, 걸은 거리 $800\,\text{m}$, 뛴 거리 $2000\,\text{m}$
- 또는 분속 $60\,\text{m}$, 분속 $200\,\text{m}$, 총 $20$분, 거리 $2.2\,\text{km}$ → $60x+200(20-x)=2200$ → $x=\frac{1800}{140}$ (여전히 분수)
- **권장**: 분속 $60\,\text{m}$, 분속 $180\,\text{m}$, 총 $20$분, 거리 $2.4\,\text{km}$ → $60x+180(20-x)=2400$ → $-120x=-1200$ → $x=10$분, 걸은 거리 $600\,\text{m}$, 뛴 거리 $1800\,\text{m}$

## 중요 이슈

### [M-01] 본문 예제 3(농도 문제)의 난이도 과다 -- "지나치게 복잡한 방정식" 해당 가능 (4-2-2_일차방정식의활용.md, 127~172행)

**문제**: $8\%$ 소금물 $300\,\text{g}$에 소금을 더 넣어 $12\%$ 소금물 만들기.

**문제점**:
1. 방정식 $\frac{24+x}{300+x} = \frac{3}{25}$에서 양변에 $25(300+x)$를 곱해야 하는데, 미지수가 포함된 식을 양변에 곱하는 것은 일차방정식의 기본 풀이 범위를 넘어선다.
2. 답이 $x = \frac{150}{11} \approx 13.6\,\text{g}$으로 분수이다.
3. 교육과정 유의사항 "지나치게 복잡한 방정식의 풀이는 다루지 않는다"에 저촉될 수 있다.

**수정안**: 소금을 더 넣는 문제 대신, **물을 더 넣는 문제**(분모에만 $x$가 포함되어 정수 답이 나옴)로 교체하거나, 수치를 조정하여 정수 답이 나오도록 설계. 예: $10\%$ 소금물 $200\,\text{g}$에 소금을 더 넣어 $20\%$ 만들기 → $\frac{20+x}{200+x}=\frac{1}{5}$ → $100+5x=200+x$ → $x=25\,\text{g}$ (정수 답)

### [M-02] 대단원 마무리 문제 13번(형/동생 따라잡기) 답이 분수 (4_대단원마무리.md, 95행)

**문제**: 형이 $10$분 먼저 분속 $80\,\text{m}$로 출발, 동생이 분속 $200\,\text{m}$로 뒤따라감. 따라잡는 시간 구하기.

**검산**: $200x = 80(x+10)$ → $120x = 800$ → $x = \frac{20}{3} \approx 6.67$분

**문제점**: 답이 $\frac{20}{3}$분으로, 정수가 아니다. 서술형 문제에서 분수 답 자체는 허용되나, 실생활 맥락에서 "$6\frac{2}{3}$분 후에 따라잡는다"는 답이 부자연스럽다. 또한 풀이 파일(unit4_solutions.md)에 이 문제의 풀이가 없다.

**수정안**: 수치 조정으로 정수 답 유도. 예: 형 분속 $60\,\text{m}$, 동생 분속 $200\,\text{m}$, $10$분 먼저 출발 → $x = \frac{600}{140}$ (여전히 분수). **권장**: 형 분속 $80\,\text{m}$, $10$분 먼저, 동생 분속 $200\,\text{m}$ → $200x = 80(x+10)$ → $\frac{20}{3}$. 대안: 형 분속 $60\,\text{m}$, 동생 분속 $180\,\text{m}$, $15$분 먼저 → $180x=60(x+15)$ → $120x=900$ → $x=7.5$분 (여전히). **최종 권장**: 형 분속 $80\,\text{m}$, 동생 분속 $120\,\text{m}$, $10$분 먼저 → $120x=80(x+10)$ → $40x=800$ → $x=20$분.

### [N-02] 문제 35번(수영장 채우기) -- 일의 양 개념의 교육과정 적절성 (unit4_problems.md, 318행)

**문제**: A 호스 6시간, B 호스 9시간. A 호스로 2시간 채운 후 B 호스로 나머지 채우기.

**문제점**: "일의 양"을 1로 놓고 단위 시간당 일의 비율($\frac{1}{6}$, $\frac{1}{9}$)로 계산하는 방식은 비례 관계의 응용이며, 중1 수준에서 직관적이지 않다. 난이도 "상"으로 표기되어 있으나, 이 유형은 많은 교과서에서 중2~3 또는 고등 수준으로 분류한다.

**수정안**: 난이도 "상"이 적절하다면 유지 가능하나, "지나치게 복잡한 방정식의 풀이는 다루지 않는다"는 교육과정 유의사항을 고려하여 삭제하거나 도전 문제로 별도 표기 권장.

### [N-03] 문제 15번(2), 15번(3), 중단원2 2번(1), 29번(4) -- 분수 해의 빈도

**문제점**: 일차방정식의 해가 분수로 나오는 문제가 다수 포함됨.

| 문제 | 답 |
|------|-----|
| 15번(2) | $x = \frac{19}{5}$ |
| 15번(3) | $x = \frac{23}{3}$ |
| 중2마무리 2번(1) | $x = \frac{21}{4}$ |
| 29번(4) | $x = \frac{21}{4}$ |

**의견**: 분수 해 자체는 교육과정 범위 내이며, 분수 계산 연습으로서 가치가 있다. 그러나 전체 비중이 높아 학생이 "답이 분수면 틀린 것 같다"는 불안감을 느낄 수 있다. 정수 해 문제와 분수 해 문제의 비율을 $7:3$ 정도로 조정하면 적절하다. 현재 비율은 대략 $6:4$.

### [N-04] 문제 40번 -- 단위 변환 과정의 복잡성 (unit4_problems.md, 350행)

**문제**: 시속 $3\,\text{km}$, 시속 $30\,\text{km}$, 총 $18$분, 거리 $4.5\,\text{km}$. 걸은 시간을 $x$**분**으로 설정.

**문제점**: 미지수를 "분" 단위로 설정했을 때 속력은 "시속(km/h)"으로 주어져 있어, 분속으로 변환하는 과정이 필수이다. 풀이에서 시속 $3\,\text{km} = \text{분속}\,\frac{1}{20}\,\text{km}$, 시속 $30\,\text{km} = \text{분속}\,\frac{1}{2}\,\text{km}$로 변환한다. 이 변환 자체가 중1 학생에게 어려울 수 있으며, 배점 $6$점 문제로서 방정식 풀이보다 단위 변환에 더 큰 인지 부담이 걸린다.

**수정안**: 속력을 처음부터 분속으로 제시하거나, 미지수를 "시간(h)" 단위로 설정하도록 유도하는 소문항 구조 조정.

### [N-05] 중단원 4-2 마무리 문제 5번 -- 방정식 세우기가 너무 쉬움

**문제**: "시속 $4\,\text{km}$로 $x$시간 걷고, 시속 $12\,\text{km}$로 $x$시간 자전거를 타서 총 $24\,\text{km}$"

**문제점**: $4x + 12x = 24$로 단순 일차방정식이며 $x = \frac{3}{2}$. 중단원 마무리 문제치고 너무 단순하다. 두 시간이 같다는 조건 때문에 미지수를 하나로 잡기가 쉬워 활용 문제의 모델링 연습이 되지 않는다.

**수정안**: 걷는 시간과 자전거 시간이 다른 조건(예: 합이 주어짐)으로 변경하면 더 의미 있는 연습이 됨.

### [N-06] 본문 예제 3(농도 문제) 풀이에서 중간 과정 "약분하고 정리하면" 설명 부족 (4-2-2_일차방정식의활용.md, 151~152행)

**현재 서술**: "양변에서 $100$을 약분하고 정리하면 $\frac{24+x}{300+x} = \frac{3}{25}$"

**문제점**: $\frac{12}{100} = \frac{3}{25}$로의 약분은 $4$로 약분한 것인데, 학생이 왜 $\frac{12}{100}$이 $\frac{3}{25}$가 되는지 혼동할 수 있다. 또한 "양변에서 $100$을 약분"이라는 표현이 수학적으로 부정확하다. 양변에서 $100$으로 나눈 것이 아니라 우변의 분수를 기약분수로 만든 것이다.

**수정안**: "$\frac{24+x}{300+x} \times 100 = 12$의 양변을 $100$으로 나누면 $\frac{24+x}{300+x} = \frac{12}{100}$. $\frac{12}{100}$을 기약분수로 나타내면 $\frac{3}{25}$이므로..."로 수정.

### [L-05] LaTeX 표기 세부 사항

- `4-1-2_등식의성질.md` 88행: `$\frac{a}{c} = \frac{b}{c}$ ($c \neq 0$)` -- 괄호 안 조건 표기가 표 내부에서 가독성이 낮음.
- `unit4_problems.md`에서 `\dfrac` 사용, 본문에서는 `\frac` 사용 -- 렌더링 결과는 동일하나 소스 일관성이 없음.

**수정안**: 분수 표기를 `\dfrac`(display style)으로 통일하거나 `\frac`으로 통일.

## 교육적 적절성 검증

### 개념 전개 순서 평가

```
등식(정의) → 방정식/항등식(구별) → 해의 의미(대입 확인)
→ 등식의 성질(양팔저울) → 등식의 성질로 풀기
→ 이항(등식 성질의 약식) → 일차방정식(정의, 표준형)
→ 일차방정식 풀이(괄호, 분수, 소수) → 활용(4단계)
```

**판정**: 논리적으로 자연스러운 전개이며, 교육과정의 의도("등식의 성질 → 이항" 순서)를 잘 반영함.

### 난이도 배분 평가

| 난이도 | 문제 수 | 비율 |
|--------|---------|------|
| 하 | 8 | 19% |
| 중하 | 5 | 12% |
| 중 | 13 | 30% |
| 중상 | 10 | 23% |
| 상 | 3 | 7% |
| 수행평가 | 4 | 9% |

**판정**: 난이도 배분이 중~중상에 집중되어 있어 적절함. "하" 비율이 약간 낮으나 본문 확인 문제가 기초 연습 역할을 보충.

### 탐구 활동 평가

- **등호의 의미 탐구** (4-1-1): 대입표 활동으로 방정식과 항등식의 차이를 발견하게 함. 수행 가능하고 효과적.
- **양팔저울 탐구** (4-1-2): 등식의 성질 4가지를 구체적 조작으로 이해하게 함. 실제 양팔저울 없이도 그림으로 수행 가능.
- **문제 만들기** (4-2-2): 학생이 직접 활용 문제를 만드는 활동. 창의성 및 수학적 사고 촉진에 좋으나, 구체적 예시가 더 있으면 좋겠음.

## 종합 의견

### 강점

1. **개념 전개의 논리적 흐름**: 등식 → 방정식 → 등식의 성질 → 이항 → 일차방정식 풀이 → 활용의 순서가 교육과정의 의도에 정확히 부합한다.
2. **검산 문화 강조**: 모든 예제와 활용 문제에서 검산을 수행하여 학생에게 좋은 습관을 형성시킨다.
3. **탐구 활동의 질**: 양팔저울 활동과 대입표 활동이 개념 이해에 효과적으로 설계되어 있다.
4. **수학적 정확성**: 본문과 풀이의 모든 수학적 계산이 정확하다.
5. **흔한 오답 분석**: 풀이에서 "흔한 오답" 코너가 교사의 지도에 유용하다.

### 개선 필요 사항

1. **[최우선]** 문제 41번의 조건 보완 또는 수정 ([C-01])
