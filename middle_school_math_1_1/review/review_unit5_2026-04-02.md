## 1. 치명적 이슈 (즉시 수정 필수)

### [치명적-1] 5-1 중단원마무리 5번: 해가 존재하지 않는 문제

- **위치**: `5-1_중단원마무리.md` 59행, 답 83행
- **문제**: "점 $\mathrm{P}(a - 2, 3a + 6)$이 원점일 때, $a$의 값을 구하시오."
- **오류**: 원점이려면 $a - 2 = 0$이고 $3a + 6 = 0$이어야 하는데, 첫 번째 조건에서 $a = 2$, 두 번째 조건에서 $a = -2$로 **연립방정식이 모순**이다. 두 조건을 동시에 만족하는 $a$는 존재하지 않는다.
- **제시된 답**: $a = 2$ (첫 번째 조건만 적용, $y$좌표 $= 3(2) + 6 = 12 \neq 0$이므로 원점이 아님)
- **수정안**: 좌표를 $(a - 2, 2a - 4)$로 변경하면 $a = 2$일 때 $(0, 0)$으로 원점이 됨. 또는 $(a - 2, a + 2)$로 변경하면 $a = 2$일 때 $(0, 4)$가 되므로 부적합하고, $(2a - 4, 3a + 6)$으로 변경하면 $a = 2$일 때 $(0, 12)$이므로 역시 부적합함. **$(a - 2, 2a - 4)$로 수정**을 권장하며, 답도 $a = 2$로 유지 가능.

### [치명적-2] unit5_solutions 33번: 계산 오류

- **위치**: `unit5_solutions.md` 594~601행
- **문제**: "정비례 $y = ax$와 반비례 $y = \frac{b}{x}$가 모두 점 $(3, -2)$를 지날 때, $a + b$의 값"
- **제시된 답**: $a + b = -\frac{8}{3}$
- **올바른 풀이**:
  - $a = \frac{-2}{3} = -\frac{2}{3}$
  - $b = 3 \times (-2) = -6$
  - $a + b = -\frac{2}{3} + (-6) = -\frac{2}{3} - \frac{18}{3} = -\frac{20}{3}$
- **수정안**: 답을 $-\frac{20}{3}$으로 수정

### [치명적-3] unit5_solutions 9번: 풀이 과정 혼란 및 초기 답 오류

- **위치**: `unit5_solutions.md` 142~161행
- **문제**: 세 점 $A(-1, 4)$, $B(5, 4)$, $C(5, -2)$로 이루어진 직각삼각형 넓이
- **오류**: 풀이에서 처음에 "$\frac{35}{2} = 17.5$"라고 적은 뒤 "정답 수정: $18$"이라고 자체 수정하고 있음. 최종 답 $18$은 올바르나, 풀이 과정에 중간 사고 과정("잠깐 --")이 노출되어 있고, 초기 오답이 삭제되지 않은 상태임.
- **올바른 답**: $\frac{1}{2} \times 6 \times 6 = 18$ (정답)
- **수정안**: 풀이에서 "$\frac{35}{2}$" 부분 및 "잠깐 --" 등 중간 사고 과정을 제거하고, 깔끔하게 $18$로 서술

## 2. 중요 이슈 (수정 필요)

### [중요-1] unit5_solutions 42번 (3): 문제 의도와 결과 불일치

- **위치**: `unit5_solutions.md` 825~857행
- **문제**: "$10\;\text{m}^2$ 전지판의 일주일 동안 **남는** 전력량"을 묻고 있으나, 결과가 $-28$ kWh (부족)
- **오류**: 문제에서 "남는 전력량"을 묻는데 답이 음수(부족)가 나옴. 풀이에서도 "수정: 계산 재확인"이라는 메모가 남아 있어 출제 의도가 불분명함.
- **수정안 A**: 문제를 "이 가정에서 전지판을 $20\;\text{m}^2$ 설치했다면"으로 변경 (이 경우 하루 발전 $16$ kWh, 주간 발전 $112$ kWh, 남는 전력 $112 - 84 = 28$ kWh)
- **수정안 B**: 문제를 "부족한 전력량은 몇 kWh인지 구하시오"로 변경
- **비고**: 서술형 문항(7점)의 (3)번에서 "남는다"라는 전제가 깨지면 학생 혼란 발생 가능

### [중요-2] 대칭이동 관련 문제의 교육과정 범위 확인 필요

- **위치**: `unit5_problems.md` 8번, 39번; `5_대단원마무리.md` 해당 없음
- **내용**: 좌표평면에서 점의 $x$축 대칭, $y$축 대칭이동은 [9수02-06] 성취기준("순서쌍과 좌표를 이해하고, 좌표평면 위에 점을 나타낼 수 있다")에 명시적으로 포함되지 않음. 교육과정 학습요소에도 "대칭이동"은 없음.
- **판단**: 좌표의 부호 변화로 다룰 수 있는 수준이므로 교과서 수준에서 활용은 가능하나, 본문에 대칭이동에 대한 설명이 전혀 없는 상태에서 문제만 출제하면 학생이 어려울 수 있음.
- **수정안**: 본문(`5-1-2_좌표평면위의점.md`)에 "참고" 또는 "생각 넓히기" 형태로 축 대칭이동 시 좌표 변화 규칙을 간략히 다루거나, 문제에 힌트를 추가할 것을 권장

## 3. 보통 이슈 (수정 권장)

### [보통-1] 반비례 확인문제 4번: 정비례/반비례가 아닌 관계

- **위치**: `5-2-2_반비례관계와그래프.md` 260~264행
- **문제**: "형과 동생이 $60$개의 구슬을 나누어 가지려고 합니다. 형이 $x$개를 가지면 동생은 $y$개를 가집니다."
- **분석**: $x + y = 60$이므로 $y = 60 - x$. 이는 정비례도 반비례도 아님.
- **문제 의도**: "(2) 이 관계는 정비례입니까, 반비례입니까?"라는 물음에 "어느 쪽도 아니다"가 정답이 되어야 함.
- **평가**: 교육적으로 유의미한 문제(반례를 통해 개념을 확인)이나, 보기에 "어느 쪽도 아닙니다"라는 선택지나 안내가 없으면 학생이 혼란을 겪을 수 있음.
- **수정안**: (2)의 물음을 "이 관계는 정비례, 반비례 중 어느 것에 해당합니까? 해당하지 않으면 그 이유를 설명하시오."로 변경

### [보통-2] 5-2 중단원마무리 8번: 답 나열의 완전성 확인

- **위치**: `5-2_중단원마무리.md` 90행, 답 110행
- **문제**: "$y$가 $x$에 반비례하고 그래프가 점 $(3, 4)$를 지납니다. $x$좌표와 $y$좌표가 모두 자연수인 점을 모두 구하시오."
- **풀이**: $a = 12$, $y = \frac{12}{x}$. $12$의 자연수 약수: $1, 2, 3, 4, 6, 12$.
- **제시된 답**: $(1, 12), (2, 6), (3, 4), (4, 3), (6, 2), (12, 1)$ -- 6개, **정확함**.
- **비고**: 문제 없음. 확인 완료.

### [보통-3] 문제 17(나)의 서술 모호성

- **위치**: `unit5_problems.md` 203행
- **문제**: "(나) $x$ km의 거리를 시속 $y$ km로 걸을 때 걸리는 시간"
- **분석**: 이 서술에서 변수가 $x$, $y$, 시간 세 가지로 보임. 시간 = $\frac{x}{y}$이므로 $x$와 $y$ 사이에 반비례 관계가 성립하지 않음(시간이 상수가 아닌 한). 풀이에서도 정확히 제외하고 있음.
- **수정안**: 더 명확하게 "(나) $x$ km의 거리를 시속 $y$ km로 걸을 때 걸리는 시간이 $2$시간이다"처럼 변수 관계를 명확히 하거나, 아예 다른 보기로 교체 권장

### [보통-4] 난이도 분포 편중

- **위치**: `unit5_problems.md` 전체
- **분석**: 전체 44문제 중 난이도 '상' 문제가 1개(42번)뿐이며, '하' 문제 8개(19%), '중하' 10개(24%), '중' 15개(36%), '중상' 6개(14%)로 상위권 학생을 위한 도전 문제가 부족함.
- **수정안**: 난이도 '상' 문제를 2~3개 추가하거나, 기존 '중상' 문제 중 일부의 난이도를 상향 조정 고려

## 7. 교육적 적절성 검증

### 7.1 개념 전개 순서

순서쌍 -> 좌표 -> 좌표평면 -> 사분면 -> 정비례 -> 반비례 -> 비교

논리적으로 적절한 전개 순서이며, 각 개념이 이전 개념 위에 자연스럽게 쌓이는 구조임.

### 7.2 공학적 도구 활용

- GeoGebra 활용 안내가 정비례(5-2-1)와 반비례(5-2-2) 모두에 포함되어 교육과정의 "공학적 도구 활용" 요구를 충족함.
- 수행평가에서도 선택적으로 GeoGebra 활용을 안내하고 있어 적절함.

### 7.3 표-식-그래프 변환

- 본문에서 표 -> 식 -> 그래프 순으로 전개하며, 세 표현 간 변환을 명시적으로 다루고 있음.
- 5-2 중단원마무리 6번에서 표-식-그래프 연결 문제가 별도로 출제되어 변환 활동을 강화함.
- 대단원 수행평가에서도 "표, 식, 그래프로 나타내시오"를 명시적으로 요구함.
