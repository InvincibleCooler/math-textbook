### 핵심 개념 총정리

#### 1. 방정식과 등식의 성질

| 개념 | 설명 |
|------|------|
| 등식 | 등호($=$)를 사용하여 두 식이 같음을 나타낸 식 |
| 방정식 | 미지수의 값에 따라 참·거짓이 결정되는 등식 |
| 항등식 | 미지수에 어떤 값을 대입하여도 항상 참인 등식 |
| 해(근) | 방정식을 참이 되게 하는 미지수의 값 |

**등식의 성질**: $a = b$이면

$$a + c = b + c, \quad a - c = b - c, \quad ac = bc, \quad \frac{a}{c} = \frac{b}{c}\;(c \neq 0)$$

#### 2. 일차방정식의 풀이

- **이항**: 등식의 한 변의 항을 부호를 바꾸어 다른 변으로 옮기는 것
- **일차방정식**: $ax + b = 0$ ($a \neq 0$) 꼴로 정리되는 방정식
- **풀이 순서**: 괄호 풀기 → 이항 → 정리 → 계수로 나누기 → 검산

#### 3. 일차방정식의 활용

$$\text{미지수 설정} \;\to\; \text{방정식 세우기} \;\to\; \text{방정식 풀기} \;\to\; \text{검산}$$

### 종합문제

#### A. 개념 확인

**1.** 다음 중 방정식인 것을 모두 고르시오.

(가) $3x + 2 = 14$ &emsp; (나) $x + 5 = 5 + x$ &emsp; (다) $9 - 3 = 6$ &emsp; (라) $\frac{x}{2} + 1 = 4$

**2.** 다음 $\square$ 안에 알맞은 것을 쓰시오.

(1) $a = b$이면 $a - 7 = $ $\square$

(2) $a = b$이면 $5a = $ $\square$

(3) $a = b$이면 $\frac{a}{3} = $ $\square$

**3.** $x = 2$가 다음 방정식의 해인지 확인하시오.

(1) $5x - 3 = 7$ &emsp; (2) $3(x + 1) = 10$

---

#### B. 일차방정식의 풀이

**4.** 다음 일차방정식을 푸시오.

(1) $4x + 7 = 19$ &emsp; (2) $3x - 8 = x + 2$

(3) $2(3x - 1) = 4(x + 3)$ &emsp; (4) $-5x + 9 = 2x - 12$

**5.** 다음 일차방정식을 푸시오.

(1) $\frac{x - 3}{2} = \frac{x + 1}{5}$ &emsp; (2) $\frac{3x + 2}{4} - \frac{x - 1}{3} = 2$

(3) $0.5x + 0.3 = 0.2x + 1.5$ &emsp; (4) $0.04x - 0.3 = 0.02x + 0.1$

---

#### C. 일차방정식의 활용

**6.** 연속하는 세 정수의 합이 $-6$일 때, 세 정수를 구하시오.

**7.** 둘레의 길이가 $40\,\text{cm}$인 직사각형에서 가로의 길이가 세로의 길이보다 $6\,\text{cm}$ 길 때, 가로와 세로의 길이를 각각 구하시오.

**8.** A 지점에서 B 지점까지 시속 $50\,\text{km}$로 가면 시속 $40\,\text{km}$로 갈 때보다 $1$시간 적게 걸립니다. A 지점에서 B 지점까지의 거리를 구하시오.

**9.** $10\,\%$의 소금물 $500\,\text{g}$에서 소금물 일부를 덜어내고 그만큼의 물을 넣었더니 $6\,\%$의 소금물이 되었습니다. 덜어낸 소금물의 양을 구하시오.

**10.** 현재 아버지의 나이는 $45$살이고, 딸의 나이는 $15$살입니다. 아버지의 나이가 딸의 나이의 $2$배가 되는 것은 몇 년 후인지 구하시오.

---

#### D. 서술형 문제

**11.** 방정식 $\frac{2x + a}{3} = x - 1$의 해가 $x = 5$일 때, 상수 $a$의 값을 구하시오. (풀이 과정을 서술하시오.)

**12.** 민지는 문구점에서 공책과 연필을 샀습니다. 공책은 한 권에 $1500$원, 연필은 한 자루에 $600$원입니다. 공책을 연필보다 $2$권 적게 사고, 전체 금액이 $9600$원이었을 때, 공책과 연필을 각각 몇 개씩 샀는지 풀이 과정을 서술하시오.

**13.** 형이 집에서 출발한 지 $10$분 후에 동생이 같은 길을 따라 출발하였습니다. 형은 분속 $80\,\text{m}$, 동생은 분속 $200\,\text{m}$로 이동한다면, 동생이 출발한 후 몇 분 후에 형을 따라잡는지 풀이 과정을 서술하시오.

### 수행평가 — 실생활 문제 해결 프로젝트

> **과제**: 일상생활에서 일차방정식으로 해결할 수 있는 문제를 찾아 보고서를 작성해 봅시다.

#### 보고서 구성

1. **문제 상황 설명**: 어떤 실생활 상황인지 구체적으로 설명합니다.
2. **미지수 설정**: 구하려는 것이 무엇인지 밝히고, $x$로 놓습니다.
3. **방정식 세우기**: 문제의 조건을 일차방정식으로 나타냅니다.
4. **방정식 풀기**: 풀이 과정을 단계별로 서술합니다.
5. **검산 및 해석**: 구한 해가 문제 상황에 적합한지 확인하고, 결과를 해석합니다.
6. **느낀 점**: 방정식이 실생활에서 어떻게 쓰이는지 소감을 적습니다.

#### 평가 기준

| 평가 항목 | 우수 | 보통 | 노력 필요 |
|-----------|------|------|-----------|
| 문제 상황이 적절한가 | 실생활과 밀접하게 연결 | 연결이 다소 약함 | 관련성이 부족 |
| 방정식을 올바르게 세웠는가 | 조건을 정확히 반영 | 일부 오류 있음 | 방정식이 부적절 |
| 풀이 과정이 논리적인가 | 단계별로 명확히 서술 | 일부 단계 누락 | 풀이 과정 미흡 |
| 검산을 하였는가 | 검산을 통해 해 확인 | 검산이 불완전 | 검산 생략 |
