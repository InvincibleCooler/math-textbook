### 탐구 활동 — 수직선 위에서 덧셈하기 (1차시)

수직선 위에서 이동을 이용하여 덧셈을 알아봅시다.

**규칙**: 양수를 더하면 오른쪽으로, 음수를 더하면 왼쪽으로 이동합니다.

[그림: 수직선 위에서 $+2$에서 출발하여 $+3$만큼 오른쪽으로 이동하여 $+5$에 도착하는 화살표]

**활동 1** $(+2) + (+3)$을 수직선 위에서 나타내 봅시다.

> $+2$에서 출발하여 오른쪽으로 $3$만큼 이동하면 $+5$에 도착합니다.
>
> 따라서 $(+2) + (+3) = +5$입니다.

**활동 2** 다음 각각의 덧셈을 수직선 위에서 나타내고 결과를 구해 봅시다.

| 식 | 수직선 이동 | 결과 |
|----|-----------|------|
| $(+3) + (+4)$ | $+3$에서 오른쪽으로 $4$ 이동 | |
| $(-2) + (-3)$ | $-2$에서 왼쪽으로 $3$ 이동 | |
| $(+5) + (-2)$ | $+5$에서 왼쪽으로 $2$ 이동 | |
| $(-4) + (+6)$ | $-4$에서 오른쪽으로 $6$ 이동 | |
| $(-3) + (+3)$ | $-3$에서 오른쪽으로 $3$ 이동 | |

[그림: 위 5개의 덧셈을 각각 수직선 위의 화살표 이동으로 나타낸 그림]

**활동 3** 위의 결과를 관찰하여 다음 빈칸을 채워 봅시다.

- 같은 부호의 두 수의 합: 두 수의 절댓값의 $\fbox{\phantom{합}}$에 공통인 부호를 붙인다.
- 다른 부호의 두 수의 합: 두 수의 절댓값의 $\fbox{\phantom{차}}$에 절댓값이 $\fbox{\phantom{큰}}$ 수의 부호를 붙인다.

> **생각해 봅시다**: 절댓값이 같고 부호가 다른 두 수의 합은 얼마인가요?

---

### 개념 정리

#### 정수의 덧셈

> **같은 부호의 두 정수의 덧셈**
>
> 두 수의 절댓값의 **합**에 공통인 부호를 붙입니다.

$$\begin{aligned}
(+3) + (+5) &= +(3 + 5) = +8 \\
(-3) + (-5) &= -(3 + 5) = -8
\end{aligned}$$

> **다른 부호의 두 정수의 덧셈**
>
> 두 수의 절댓값의 **차**에 절댓값이 큰 수의 부호를 붙입니다.

$$\begin{aligned}
(+7) + (-3) &= +(7 - 3) = +4 \\
(-7) + (+3) &= -(7 - 3) = -4
\end{aligned}$$

> **절댓값이 같고 부호가 다른 두 수의 덧셈**

$$(+5) + (-5) = 0, \qquad (-3) + (+3) = 0$$

> **$0$과의 덧셈**

$$(+4) + 0 = +4, \qquad 0 + (-7) = -7$$

어떤 수에 $0$을 더하면 그 수 자체가 됩니다.

### 예제 1

다음을 계산하시오.

**(1)** $(+8) + (+6)$ $\qquad$ **(2)** $(-5) + (-9)$

**(3)** $(+12) + (-7)$ $\qquad$ **(4)** $(-10) + (+4)$

**풀이**

**(1)** $(+8) + (+6) = +(8 + 6) = +14$

> **[왜?]** 같은 부호(양수)끼리의 덧셈이므로 절댓값의 합 $8 + 6 = 14$에 $+$ 부호를 붙입니다.

**(2)** $(-5) + (-9) = -(5 + 9) = -14$

**(3)** $(+12) + (-7) = +(12 - 7) = +5$

> **[왜?]** 다른 부호의 덧셈이므로 절댓값의 차 $12 - 7 = 5$에 절댓값이 큰 $+12$의 부호인 $+$를 붙입니다.

**(4)** $(-10) + (+4) = -(10 - 4) = -6$

### 개념 정리

#### 유리수의 덧셈

정수의 덧셈과 같은 방법으로, 유리수의 덧셈도 부호와 절댓값을 이용하여 계산합니다.

### 예제 2

다음을 계산하시오.

**(1)** $\left(-\frac{2}{3}\right) + \left(-\frac{4}{3}\right)$ $\qquad$ **(2)** $(+1.5) + (-3.7)$

**(3)** $\left(-\frac{3}{4}\right) + \left(+\frac{5}{6}\right)$

**풀이**

**(1)** $\left(-\frac{2}{3}\right) + \left(-\frac{4}{3}\right) = -\left(\frac{2}{3} + \frac{4}{3}\right) = -\frac{6}{3} = -2$

**(2)** $(+1.5) + (-3.7) = -(3.7 - 1.5) = -2.2$

> **[왜?]** 절댓값을 비교하면 $3.7 > 1.5$이므로 절댓값이 큰 $-3.7$의 부호 $-$를 붙이고, 절댓값의 차 $3.7 - 1.5 = 2.2$를 구합니다.

**(3)** $\left(-\frac{3}{4}\right) + \left(+\frac{5}{6}\right) = +\left(\frac{5}{6} - \frac{3}{4}\right) = +\left(\frac{10}{12} - \frac{9}{12}\right) = +\frac{1}{12} = \frac{1}{12}$

### 개념 정리

#### 덧셈의 교환법칙과 결합법칙

정수와 유리수의 덧셈에서도 자연수의 덧셈과 마찬가지로 다음 법칙이 성립합니다.

> **덧셈의 교환법칙(交換法則)**
>
> 두 수의 순서를 바꾸어 더해도 합은 같습니다.
>
> $$a + b = b + a$$

> **덧셈의 결합법칙(結合法則)**
>
> 세 수의 덧셈에서 앞의 두 수를 먼저 더해도, 뒤의 두 수를 먼저 더해도 합은 같습니다.
>
> $$(a + b) + c = a + (b + c)$$

이 법칙들을 활용하면 계산을 편리하게 할 수 있습니다.

### 예제 3

다음을 계산하시오.

$$(-3) + (+7) + (-5) + (+1)$$

**풀이**

교환법칙과 결합법칙을 이용하여, 양수끼리 음수끼리 모아서 계산하면 편리합니다.

$$\begin{aligned}
& (-3) + (+7) + (-5) + (+1) \\
&= \{(+7) + (+1)\} + \{(-3) + (-5)\} \\
&= (+8) + (-8) \\
&= 0
\end{aligned}$$

### 개념 정리 (2차시)

#### 정수와 유리수의 뺄셈

뺄셈은 빼는 수의 부호를 바꾸어 덧셈으로 바꿀 수 있습니다.

> **뺄셈을 덧셈으로 바꾸기**
>
> $$a - b = a + (-b)$$
>
> 즉, 어떤 수를 빼는 것은 그 수의 **부호를 바꾸어 더하는 것**과 같습니다.

이 원리를 확인해 봅시다.

$$\begin{aligned}
(+5) - (+3) &= (+5) + (-3) = +2 \\
(+5) - (-3) &= (+5) + (+3) = +8 \\
(-5) - (+3) &= (-5) + (-3) = -8 \\
(-5) - (-3) &= (-5) + (+3) = -2
\end{aligned}$$

> **[왜?]** $(+5) - (-3) = +8$인 이유를 생각해 봅시다. $\square + (-3) = +5$에서 $\square = +8$이므로 $(+5) - (-3) = +8$입니다. 이는 $(-3)$의 부호를 바꾸어 $(+5) + (+3) = +8$로 계산한 것과 같습니다.

### 개념 정리

#### 덧셈과 뺄셈의 혼합 계산

덧셈과 뺄셈이 섞인 식은 뺄셈을 모두 덧셈으로 바꾸어 계산합니다. 이때 괄호와 $+$ 기호를 생략하여 간단히 나타낼 수 있습니다.

### 예제 6

다음을 계산하시오.

$$(+5) - (-3) + (-7) - (+2)$$

**풀이**

$$\begin{aligned}
& (+5) - (-3) + (-7) - (+2) \\
&= (+5) + (+3) + (-7) + (-2) \quad \cdots \text{뺄셈을 덧셈으로 변환} \\
&= 5 + 3 + (-7) + (-2) \quad \cdots \text{괄호와 }+\text{ 기호 생략} \\
&= 5 + 3 - 7 - 2 \quad \cdots \text{간단히 나타내기} \\
&= (5 + 3) + (-7 - 2) \quad \cdots \text{양수끼리, 음수끼리 모으기} \\
&= 8 + (-9) \\
&= -1
\end{aligned}$$

> **참고** 위 계산에서 $5 + 3 - 7 - 2$처럼 괄호를 모두 생략한 식에서, $-7$은 '$7$을 빼라'가 아니라 '$-7$을 더하라'는 뜻입니다.

### 예제 7 (3차시 — 연습)

다음을 계산하시오.

**(1)** $7 - 12 + 3 - 8$

**(2)** $-\frac{5}{6} + \frac{3}{4} - \frac{1}{3}$

**(3)** $-2.5 + 4.3 - 1.8 + 0.5$

**풀이**

**(1)**
$$\begin{aligned}
7 - 12 + 3 - 8 &= (7 + 3) + (-12 - 8) \\
&= 10 + (-20) = -10
\end{aligned}$$

**(2)** 통분하여 계산합니다. 분모의 최소공배수는 $12$입니다.
$$\begin{aligned}
-\frac{5}{6} + \frac{3}{4} - \frac{1}{3} &= -\frac{10}{12} + \frac{9}{12} - \frac{4}{12} \\
&= \frac{-10 + 9 - 4}{12} = \frac{-5}{12} = -\frac{5}{12}
\end{aligned}$$

**(3)**
$$\begin{aligned}
-2.5 + 4.3 - 1.8 + 0.5 &= (4.3 + 0.5) + (-2.5 - 1.8) \\
&= 4.8 + (-4.3) = 0.5
\end{aligned}$$
