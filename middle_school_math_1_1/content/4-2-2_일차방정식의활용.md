> **학습 목표**: 실생활 문제를 일차방정식으로 모델링하여 해결할 수 있다.

### 탐구 활동 — 문제 해결의 단계

다음 문제를 읽고, 방정식을 세워 해결하는 과정을 함께 알아봅시다.

> 어떤 수의 $3$배에서 $4$를 뺀 값이 $11$이다. 어떤 수를 구하시오.

**단계 1: 미지수 설정**

구하려는 값을 $x$로 놓습니다.

어떤 수를 $x$로 놓으면,

**단계 2: 방정식 세우기**

문제의 조건을 식으로 나타냅니다.

$$3x - 4 = 11$$

**단계 3: 방정식 풀기**

$$3x = 11 + 4 = 15$$

$$x = 5$$

**단계 4: 검산**

구한 값이 문제의 조건에 맞는지 확인합니다.

$5$의 $3$배에서 $4$를 빼면 $15 - 4 = 11$이므로 조건에 맞습니다. ✓

따라서 어떤 수는 $5$입니다.

---

### 개념 정리

#### 일차방정식의 활용 과정

일차방정식을 활용하여 문제를 해결할 때는 다음 네 단계를 따릅니다.

```
미지수 설정 → 방정식 세우기 → 방정식 풀기 → 검산
```

| 단계 | 내용 |
|------|------|
| 미지수 설정 | 구하려는 것을 $x$로 놓고, $x$가 나타내는 것을 분명히 합니다. |
| 방정식 세우기 | 문제의 조건을 $x$를 사용한 등식으로 나타냅니다. |
| 방정식 풀기 | 일차방정식의 풀이 방법으로 $x$의 값을 구합니다. |
| 검산 | 구한 값이 문제의 조건에 맞는지 확인합니다. |

> **주의**: 방정식의 해가 문제의 상황에 적합한지 반드시 확인해야 합니다. 예를 들어 사람 수를 구하는 문제에서 해가 음수나 소수가 나오면 문제의 조건을 다시 살펴보아야 합니다.

### 예제 1 — 수에 대한 문제

연속하는 세 홀수의 합이 $51$일 때, 세 홀수를 구하시오.

**풀이**

**미지수 설정**: 가운데 홀수를 $x$로 놓으면, 연속하는 세 홀수는 $x - 2$, $x$, $x + 2$입니다.

**방정식 세우기**:

$$(x - 2) + x + (x + 2) = 51$$

**방정식 풀기**:

$$3x = 51$$

$$x = 17$$

**검산**: 세 홀수는 $15$, $17$, $19$이고, $15 + 17 + 19 = 51$ ✓

따라서 세 홀수는 $15$, $17$, $19$입니다.

### 예제 2 — 거리, 속력, 시간 문제

> **참고**: (거리) $=$ (속력) $\times$ (시간), 즉 $d = vt$

민수는 집에서 도서관까지 걸어서 가려고 합니다. 시속 $4\,\text{km}$로 걸으면 시속 $6\,\text{km}$로 걸을 때보다 $30$분 더 걸립니다. 집에서 도서관까지의 거리를 구하시오.

**풀이**

**미지수 설정**: 집에서 도서관까지의 거리를 $x\,\text{km}$로 놓습니다.

**방정식 세우기**:

시속 $4\,\text{km}$로 걸을 때 걸리는 시간: $\frac{x}{4}$ 시간

시속 $6\,\text{km}$로 걸을 때 걸리는 시간: $\frac{x}{6}$ 시간

$30$분 $= \frac{1}{2}$시간이므로

$$\frac{x}{4} - \frac{x}{6} = \frac{1}{2}$$

> [왜?] 느린 속력으로 걸을 때 시간이 더 많이 걸리므로, (느린 쪽 시간) $-$ (빠른 쪽 시간) $=$ 차이 시간으로 식을 세웁니다.

**방정식 풀기**:

양변에 $12$를 곱하면

$$3x - 2x = 6$$

$$x = 6$$

**검산**: 시속 $4\,\text{km}$로 $6\,\text{km}$를 가면 $\frac{6}{4} = \frac{3}{2}$시간, 시속 $6\,\text{km}$로 가면 $\frac{6}{6} = 1$시간이므로 차이는 $\frac{1}{2}$시간 $= 30$분 ✓

따라서 집에서 도서관까지의 거리는 $6\,\text{km}$입니다.

### 예제 3 — 농도 문제

> **참고**: (소금물의 농도) $= \frac{\text{(소금의 양)}}{\text{(소금물의 양)}} \times 100$ (%)

$8\,\%$의 소금물 $300\,\text{g}$에 소금을 더 넣어 $12\,\%$의 소금물을 만들려고 합니다. 소금을 몇 $\text{g}$ 더 넣어야 합니까?

**풀이**

**미지수 설정**: 더 넣을 소금의 양을 $x\,\text{g}$으로 놓습니다.

**방정식 세우기**:

처음 소금의 양: $300 \times \frac{8}{100} = 24\,\text{g}$

소금을 $x\,\text{g}$ 더 넣으면

- 소금의 양: $(24 + x)\,\text{g}$
- 소금물의 양: $(300 + x)\,\text{g}$

농도가 $12\,\%$이므로

$$\frac{24 + x}{300 + x} \times 100 = 12$$

**방정식 풀기**:

양변에서 $100$을 약분하고 정리하면

$$\frac{24 + x}{300 + x} = \frac{12}{100} = \frac{3}{25}$$

양변에 $25(300 + x)$를 곱하면

$$25(24 + x) = 3(300 + x)$$

$$600 + 25x = 900 + 3x$$

$$25x - 3x = 900 - 600$$

$$22x = 300$$

$$x = \frac{300}{22} = \frac{150}{11} \approx 13.6$$

**검산**: 소금 $\frac{150}{11}\,\text{g}$을 더 넣으면, 소금 $= 24 + \frac{150}{11} = \frac{414}{11}\,\text{g}$, 소금물 $= 300 + \frac{150}{11} = \frac{3450}{11}\,\text{g}$

농도 $= \frac{414/11}{3450/11} \times 100 = \frac{414}{3450} \times 100 = 12\,\%$ ✓

따라서 소금을 $\frac{150}{11}\,\text{g}$ (약 $13.6\,\text{g}$) 더 넣어야 합니다.

### 예제 4 — 가격 문제

한 자루에 $800$원인 볼펜과 한 자루에 $1200$원인 샤프를 합하여 $10$자루를 사고 $10000$원을 냈더니 거스름돈이 $800$원이었습니다. 볼펜과 샤프를 각각 몇 자루씩 샀습니까?

**풀이**

**미지수 설정**: 볼펜을 $x$자루 샀다고 놓으면, 샤프는 $(10 - x)$자루입니다.

**방정식 세우기**:

지불한 금액 $= 10000 - 800 = 9200$원이므로

$$800x + 1200(10 - x) = 9200$$

**방정식 풀기**:

$$800x + 12000 - 1200x = 9200$$

$$-400x = 9200 - 12000$$

$$-400x = -2800$$

$$x = 7$$

**검산**: 볼펜 $7$자루: $800 \times 7 = 5600$원, 샤프 $3$자루: $1200 \times 3 = 3600$원, 합계 $= 9200$원 ✓

따라서 볼펜은 $7$자루, 샤프는 $3$자루 샀습니다.

### 예제 5 — 나이 문제

현재 어머니의 나이는 $42$살이고, 아들의 나이는 $12$살입니다. 어머니의 나이가 아들의 나이의 $2$배가 되는 것은 몇 년 후입니까?

**풀이**

**미지수 설정**: $x$년 후라고 놓습니다.

**방정식 세우기**:

$x$년 후 어머니의 나이: $(42 + x)$살

$x$년 후 아들의 나이: $(12 + x)$살

어머니의 나이가 아들 나이의 $2$배이므로

$$42 + x = 2(12 + x)$$

**방정식 풀기**:

$$42 + x = 24 + 2x$$

$$42 - 24 = 2x - x$$

$$x = 18$$

**검산**: $18$년 후 어머니 $= 60$살, 아들 $= 30$살이고, $60 = 2 \times 30$ ✓

따라서 $18$년 후입니다.
