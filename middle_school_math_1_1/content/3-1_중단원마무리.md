# 1. 문자의 사용과 식의 값

## 중단원 마무리

> **학습 목표**: 문자의 사용과 식의 값을 종합 정리한다.  
> **성취기준**: [9수02-01], [9수02-02]

---

### 핵심 개념 요약

| 개념 | 내용 |
|------|------|
| **문자식** | 수량 관계를 문자를 사용하여 나타낸 식 |
| **곱셈 기호 생략** | $\times$를 생략하고, 수를 문자 앞에 쓴다. 같은 문자의 곱은 거듭제곱으로 나타낸다. |
| **나눗셈의 분수 표현** | $\div$를 사용하지 않고 분수 형태로 나타낸다. |
| **대입** | 문자에 특정한 수를 넣는 것 |
| **식의 값** | 문자에 수를 대입하여 계산한 결과 |

---

### 연습문제

**1.** 다음 식에서 곱셈 기호와 나눗셈 기호를 생략하여 나타내시오.

$\quad$ (1) $a \times (-1) \times b$ $\qquad$ (2) $x \times x \times y \times 2$ $\qquad$ (3) $(a - b) \div 5$

**2.** 다음 수량을 문자를 사용한 식으로 나타내시오.

$\quad$ (1) $x$km의 거리를 시속 $4\,$km로 걸을 때 걸리는 시간

$\quad$ (2) $a$개의 사탕을 $7$명이 똑같이 나누었을 때 한 사람이 받는 사탕의 수

$\quad$ (3) 가로의 길이가 $x\,$cm이고 세로의 길이가 가로의 길이보다 $3\,$cm 더 긴 직사각형의 둘레의 길이

**3.** 다음 식을 곱셈 기호와 나눗셈 기호를 사용하여 나타내시오.

$\quad$ (1) $4a^2b$ $\qquad$ (2) $\dfrac{3x + y}{2}$

**4.** $x = -2$일 때, 다음 식의 값을 구하시오.

$\quad$ (1) $3x + 10$ $\qquad$ (2) $-x^2 + 3x$ $\qquad$ (3) $\dfrac{x^2 - 1}{x + 2}$

**5.** $a = 3$, $b = -2$일 때, 다음 식의 값을 구하시오.

$\quad$ (1) $a^2 - b^2$ $\qquad$ (2) $2ab + b$ $\qquad$ (3) $\dfrac{a - b}{a + b}$

**6.** [서술형] 정사각형의 한 변의 길이를 $a\,$cm라 할 때, 다음 물음에 답하시오.

$\quad$ (1) 정사각형의 둘레의 길이를 $a$를 사용한 식으로 나타내시오.

$\quad$ (2) 정사각형의 넓이를 $a$를 사용한 식으로 나타내시오.

$\quad$ (3) $a = 5$일 때, 둘레의 길이와 넓이를 각각 구하시오.

**7.** [서술형] 어떤 택시의 기본요금은 $4800$원이고, 기본거리를 초과한 후 $1\,$km를 이동할 때마다 $1000$원씩 추가됩니다. 기본거리를 초과한 후 $d\,$km를 더 이동했을 때, 택시 요금을 $d$를 사용한 식으로 나타내고, $d = 7$일 때 택시 요금을 구하시오.

### 정답

**1.** (1) $-ab$ $\quad$ (2) $2x^2y$ $\quad$ (3) $\dfrac{a-b}{5}$

**2.** (1) $\dfrac{x}{4}$시간 $\quad$ (2) $\dfrac{a}{7}$개 $\quad$ (3) $(4x + 6)\,$cm

**3.** (1) $4 \times a \times a \times b$ $\quad$ (2) $(3 \times x + y) \div 2$

**4.** (1) $4$ $\quad$ (2) $-10$ $\quad$ (3) 식의 값이 존재하지 않는다. ($x + 2 = 0$이므로)

> **주의** 4.(3)에서 $x = -2$를 대입하면 분모가 $0$이 되므로, 이 경우 식의 값을 구할 수 없습니다.

**5.** (1) $5$ $\quad$ (2) $-14$ $\quad$ (3) $5$

**6.** (1) $4a\,$cm $\quad$ (2) $a^2\,$cm$^2$ $\quad$ (3) 둘레의 길이: $20\,$cm, 넓이: $25\,$cm$^2$

**7.** 택시 요금: $(4800 + 1000d)$원, $d = 7$일 때: $4800 + 1000 \times 7 = 11800$(원)
