### 핵심 개념 정리

#### 1. 덧셈과 뺄셈

| 개념 | 내용 |
|------|------|
| 같은 부호의 덧셈 | 절댓값의 합에 공통 부호를 붙인다 |
| 다른 부호의 덧셈 | 절댓값의 차에 절댓값이 큰 수의 부호를 붙인다 |
| 뺄셈 | 빼는 수의 부호를 바꾸어 덧셈으로 바꾼다: $a - b = a + (-b)$ |

#### 2. 곱셈과 나눗셈

| 개념 | 내용 |
|------|------|
| 같은 부호의 곱셈 | 절댓값의 곱에 $+$ 부호를 붙인다 |
| 다른 부호의 곱셈 | 절댓값의 곱에 $-$ 부호를 붙인다 |
| 세 수 이상의 곱셈 | 음수의 개수가 짝수이면 $+$, 홀수이면 $-$ |
| 역수 | 곱해서 $1$이 되는 수. $\frac{a}{b}$의 역수는 $\frac{b}{a}$ ($0$의 역수는 없다) |
| 나눗셈 | 나누는 수의 역수를 곱한다: $a \div b = a \times \frac{1}{b}$ |

#### 3. 사칙 혼합 계산

| 순서 | 내용 |
|------|------|
| 1 | 괄호 안을 먼저 계산한다 |
| 2 | 거듭제곱을 계산한다 |
| 3 | 곱셈, 나눗셈을 왼쪽부터 차례로 계산한다 |
| 4 | 덧셈, 뺄셈을 왼쪽부터 차례로 계산한다 |

#### 4. 연산 법칙

| 법칙 | 덧셈 | 곱셈 |
|------|------|------|
| 교환법칙 | $a + b = b + a$ | $a \times b = b \times a$ |
| 결합법칙 | $(a + b) + c = a + (b + c)$ | $(a \times b) \times c = a \times (b \times c)$ |
| 분배법칙 | $a \times (b + c) = a \times b + a \times c$ | |

### 연습문제

**1.** 다음을 계산하시오.

$\quad$ (1) $(+7) + (-12)$ $\qquad$ (2) $(-9) + (-6)$

$\quad$ (3) $(+8) - (+13)$ $\qquad$ (4) $(-5) - (-11)$

---

**2.** 다음을 계산하시오.

$\quad$ (1) $\left(-\frac{3}{4}\right) + \left(+\frac{5}{8}\right)$ $\qquad$ (2) $(+2.7) - (-1.6)$

$\quad$ (3) $\left(-\frac{2}{3}\right) - \left(+\frac{1}{4}\right)$ $\qquad$ (4) $(-0.8) + (-3.5) - (-1.2)$

---

**3.** 다음을 계산하시오.

$\quad$ (1) $(-6) \times (+7)$ $\qquad$ (2) $(-8) \times (-9)$

$\quad$ (3) $(+24) \div (-6)$ $\qquad$ (4) $(-36) \div (-4)$

---

**4.** 다음을 계산하시오.

$\quad$ (1) $\left(-\frac{3}{5}\right) \times \left(+\frac{10}{9}\right)$ $\qquad$ (2) $\left(+\frac{7}{8}\right) \div \left(-\frac{3}{4}\right)$

$\quad$ (3) $(-0.3) \times (-0.5) \times (+4)$ $\qquad$ (4) $\left(-\frac{2}{3}\right) \div \left(+\frac{4}{9}\right) \times \left(-\frac{1}{2}\right)$

**5.** 다음 수의 역수를 구하시오.

$\quad$ (1) $-\frac{5}{3}$ $\qquad$ (2) $7$ $\qquad$ (3) $-0.4$ $\qquad$ (4) $-1$

**6.** 다음을 계산하시오.

**7.** 다음을 계산하시오.

**8.** 분배법칙을 이용하여 다음을 계산하시오.
