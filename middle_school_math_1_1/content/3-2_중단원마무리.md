# 2. 일차식의 계산

## 중단원 마무리

> **학습 목표**: 일차식의 계산을 종합 정리한다.  
> **성취기준**: [9수02-03]

---

### 핵심 개념 요약

| 개념 | 내용 |
|------|------|
| **항** | 식에서 덧셈 기호 $+$로 연결된 각각의 부분 |
| **상수항** | 문자를 포함하지 않는 항 |
| **계수** | 문자에 곱해진 수 |
| **차수** | 문자가 곱해진 횟수 |
| **일차식** | 차수가 $1$인 다항식 |
| **동류항** | 문자와 차수가 모두 같은 항 |
| **분배법칙** | $(a + b) \times c = ac + bc$ |
| **일차식 $\times$ 수** | 분배법칙을 이용하여 각 항에 수를 곱한다. |
| **일차식 $\div$ 수** | 나누는 수의 역수를 곱하여 계산한다. |
| **일차식의 덧셈** | 괄호를 풀고 동류항끼리 모아 계산한다. |
| **일차식의 뺄셈** | 빼는 식의 각 항의 부호를 바꾸어 더한다. |

---

### 연습문제

**1.** 다음 식에서 항, 상수항, $x$의 계수를 각각 구하시오.

$\quad$ (1) $-3x + 8$ $\qquad$ (2) $\dfrac{x}{5} - 1$ $\qquad$ (3) $7 - 2x$

**2.** 다음 중 $x$에 대한 일차식인 것을 모두 고르시오.

$\quad$ (가) $x + 4$ $\qquad$ (나) $x^2 - 3x$ $\qquad$ (다) $-5$ $\qquad$ (라) $\dfrac{2x}{3} + 1$ $\qquad$ (마) $3x^2$

**3.** 다음을 계산하시오.

$\quad$ (1) $4(2x + 3)$ $\qquad$ (2) $-3(x - 6)$ $\qquad$ (3) $(12x - 8) \div 4$

$\quad$ (4) $\left(-\dfrac{3}{2}x + 9\right) \times \left(-\dfrac{2}{3}\right)$ $\qquad$ (5) $(15x - 10) \div \left(-\dfrac{5}{3}\right)$

**4.** 다음 식을 동류항끼리 모아 간단히 하시오.

$\quad$ (1) $4x - 3 + 2x + 7$ $\qquad$ (2) $-x + 8 + 5x - 11$

**5.** 다음을 계산하시오.

$\quad$ (1) $(5x + 2) + (3x - 9)$ $\qquad$ (2) $(4x - 7) - (x + 3)$

$\quad$ (3) $3(x + 2) + 2(4x - 5)$ $\qquad$ (4) $5(2x - 1) - 3(4x - 3)$

**6.** 다음을 계산하시오.

$\quad$ (1) $\dfrac{3x - 1}{2} + \dfrac{x + 5}{4}$ $\qquad$ (2) $\dfrac{2x + 3}{3} - \dfrac{x - 2}{6}$

**7.** [서술형] 어떤 일차식 $A$에 대하여 $A + (3x - 2) = 7x + 5$일 때, 일차식 $A$를 구하시오. 풀이 과정을 쓰시오.

**8.** [서술형] 가로의 길이가 $(5x - 2)\,$cm, 세로의 길이가 $(2x + 3)\,$cm인 직사각형이 있습니다.

$\quad$ (1) 이 직사각형의 둘레의 길이를 $x$에 대한 일차식으로 나타내시오.

$\quad$ (2) $x = 4$일 때, 둘레의 길이를 구하시오.
