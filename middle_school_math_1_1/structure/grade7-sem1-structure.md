| 13주 | 47~50 | IV. 일차방정식 | 활용 (1)~(2) + 활용 심화 + 중단원 마무리 2 |
| 14주 | 51~52, 53~54 | IV → V | 대단원 마무리 + 좌표평면 도입 + 순서쌍과 좌표 + 사분면 |
| 15주 | 55~58 | V. 좌표평면과 그래프 | 중단원 마무리 1 + 정비례 (1)~(2) + 반비례 (1) |

## 설계 근거

### 차시 배분 근거

| 대단원 | 배정 차시 | 근거 |
|--------|----------|------|
| I. 소인수분해 | 10차시 | 성취기준 2개, 소인수분해는 초등 약수/배수의 확장으로 비교적 적은 차시 필요 |
| II. 정수와 유리수 | 16차시 | 성취기준 3개, 수의 확장(음수 도입) + 사칙계산 4종류로 충분한 연습 시간 필요 |
| III. 문자의 사용과 식의 계산 | 12차시 | 성취기준 3개, 문자식이 처음 도입되어 개념 이해에 시간 필요하나 계산 자체는 단순 |
| IV. 일차방정식 | 14차시 | 성취기준 2개이나 활용 문제의 비중이 크고 다양한 맥락에서 연습 필요 |
| V. 좌표평면과 그래프 | 12차시 | 성취기준 2개, 정비례/반비례의 표/식/그래프 전환 학습에 충분한 시간 배정 |

### 선수학습 연계

- **I단원**: 초등 약수/배수 -> 소인수분해로 체계화
- **II단원**: 초등 분수/소수 사칙연산 -> 음수 범위로 확장 (I단원의 소인수분해 활용)
- **III단원**: II단원의 유리수 -> 문자로 일반화 (수에서 문자로의 전환)
- **IV단원**: III단원의 일차식 -> 등식/방정식으로 확장 (식의 계산 능력 전제)
- **V단원**: II단원의 수직선 -> 좌표평면으로 확장, IV단원의 방정식 -> 그래프로 시각화
